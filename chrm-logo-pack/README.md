# CHRM Logo Pack

---

### The Logo

The CHRM "Constellation" logo represents the centre as a hub connecting different methodological approaches and research domains. The four satellite elements (squares and circles) represent the diversity of methods — from structured trials methodology to organic evidence synthesis — all unified at the central hub.

**Design elements:**
- **Central teal hub** — CHRM as the core
- **Pink square** (top-left, rotated) — represents structured methodology/TMRN
- **Blue circle** (top-right) — represents synthesis/ESI  
- **Orange square** (bottom-left) — represents diverse approaches
- **Teal circle** (bottom-right) — represents connected outcomes
- **Connecting lines** — the network that links everything

### Usage Guidelines

#### Primary Logo
Use `chrm-logo-primary` for most applications:
- Website header
- Documents and reports
- Presentations
- Email signatures
- Letterheads

#### Full Name Version
Use `chrm-logo-fullname` when:
- The abbreviation CHRM needs clarification
- Formal/official documents
- First introduction of the brand

#### Stacked Logo
Use `chrm-logo-stacked` when:
- Horizontal space is limited
- Square or portrait formats
- Social media profile pictures

#### Icon Only
Use `chrm-icon` for:
- Favicons
- App icons
- Small UI elements
- Watermarks
- Profile pictures

#### Monochrome
Use `chrm-logo-mono` for:
- Single-colour printing
- Fax headers
- Formal documents
- Embossing/engraving

#### Reversed
Use `chrm-logo-reversed` for:
- Dark backgrounds
- Photo overlays
- Video
- Coloured backgrounds

### Typography

**Primary font:** Helvetica Neue Bold (or Arial Bold as fallback)

**Text specifications:**
- "CHRM" — Helvetica Neue Bold, #1a1a1a
- "Health Research Methodology" — Helvetica Neue Regular, #999999

**Web-safe alternatives:**
- Arial Bold / Arial Regular
- system-ui
- -apple-system, BlinkMacSystemFont

**CSS example:**
```css
font-family: "Helvetica Neue", Arial, sans-serif;
```

### Minimum Sizes

- **Primary horizontal logo:** minimum 150px wide
- **Stacked logo:** minimum 80px wide  
- **Icon only:** minimum 32px
- **Favicon:** 16px minimum

---

### Clear Space

Maintain clear space around the logo equal to the diameter of the central hub circle (approximately 28% of the logo mark height).

---

### Don'ts

❌ Don't rotate the logo  
❌ Don't change the colours  
❌ Don't add effects (shadows, gradients, outlines)  
❌ Don't stretch or distort  
❌ Don't rearrange the satellite elements  
❌ Don't place on busy backgrounds without sufficient contrast  
❌ Don't use the icon without the connecting lines  
❌ Don't recreate the logo in different proportions  

---

### File Formats

**SVG (Scalable Vector Graphics)**
- Use for: web, presentations, print at any size
- Benefits: infinitely scalable, small file size, editable

**PNG (Portable Network Graphics)**
- Use for: when SVG is not supported, social media, email
- @2x versions: for retina/high-DPI displays
- @3x versions: for very high resolution needs
